# Prerequisites
* Claude Code
* NodeJS

# How to use

## Find PLPs
Use prompt to find PLPs, e.g. "Find PLPs for all websites in EXAMPLES.md".
This uses the plp-finder skill and creates a report per website in the reports folder, e.g. reports/beterbed-plps.md

## Run performance test
Run performance test with prompt, e.g.: "Run lighthouse on PLPs from reports/futurumshop-plps.md and https://www.futurumshop.nl/fietsonderhoud"
This creates a report in the reports folder, e.g. reports/plp-lighthouse-beterbed-2026-03-10.md
